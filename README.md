## 🗂️ Como baixar e iniciar o projeto 

```bash

    #clonar o projeto
    $ git clone https://github.com/saviofreitass/DataB-FrontEnd.git

    #entrar no diretório
    $ cd DataB-FrontEnd

    #instalar as dependências
    $ npm install

    #iniciar a aplicação
    $ npm run dev
```
